## Form structure

### Section 1 — Founder

| Field | Type |
| --- | --- |
| Name (and co-founder names if any) | short text |
| Email | email |
| LinkedIn URL | url |
| X / Twitter handle | short text |
| GitHub URL | url (optional) |
| Co-founder history (how long known, why working together) | long text (skip if solo) |

### Section 2 — Background

| Field | Type |
| --- | --- |
| What have you built or achieved in the past that you're most proud of? | long text |

### Section 3 — Project

| Field | Type |
| --- | --- |
| Project name | short text |
| What problem are you solving? | long text |
| Why are you solving that problem? | long text |
| Why is it urgent / unavoidable for you specifically? | long text (conditional follow-up) |

### Section 4 — Moat

| Field | Type |
| --- | --- |
| Why is your project better than competitors and what is your moat? | long text |

### Section 5 — Traction

| Field | Type |
| --- | --- |
| Current traction (waitlist, MRR, users, anything) | long text |

### Section 6 — Funding & goals

| Field | Type |
| --- | --- |
| Amount raised to date in USD, and from whom | short text |
| What do you plan to achieve by October 26, 2026? | long text |

### Section 7 — Source

| Field | Type |
| --- | --- |
| How did you learn about cyber•Fund? | short text |
