# 04-applications — Spec

**What this lane is:** Blank form templates. One file per program, unfilled.

**In the app:** The template library. When a user starts a new application, they pick a program and get a pre-structured blank pulled from here.

**What lives here:** Bare forms only — question fields present, answers empty.
**What does NOT live here:** Working drafts (→ `07-apply/`), submitted records (→ `08-submitted/`), answer content (→ `06-answers/`).
