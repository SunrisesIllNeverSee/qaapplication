# 07-apply/need-to-apply

Use this folder when an application has already been seeded into the system
but has not yet been actively worked through or filed.

Meaning:
- the entity record exists in `03-programs/`
- the bare form exists in `04-applications/`
- the question source exists in `05-questions/source/`
- the application is known and staged
- but it is still waiting to be actively applied for

## How to use it

- Keep the live apply packet in `../<slug>.md` once drafting starts.
- Use this folder for lightweight queue markers, notes, or staging documents
  that mean "this is in the system, but not yet being actively pushed."
- Do not treat this folder as submitted state.

## Current intent

This folder exists to distinguish:
- seeded but not yet actively applying
- actively drafting / applying
- already submitted

Pair this folder with `../../tracker.md` for the current operator view.
