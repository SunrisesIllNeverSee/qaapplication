# 07-answers — Spec

**What this lane is:** The rolling answer bank. Every answer ever written, indexed by question code, reusable across programs.

**In the app:** The Answer Bank. The core compounding asset — the more applications a user files, the richer this gets. AI drafting pulls from here first before generating new content.

**What lives here:** `archive/` (answers by program), `index/` (answers by theme/question code), `archived_applications/` (full historical records).
