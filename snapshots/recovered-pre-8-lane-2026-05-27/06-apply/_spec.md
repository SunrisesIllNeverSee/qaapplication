# 06-apply — Spec

**What this lane is:** Active application workstation. One file per program being actively worked.

**In the app:** The drafting workspace. The user sees the form pre-filled from their answer bank, edits in place, iterates. This is where the AI drafting and answer-bank pull happens live.

**What lives here:** Working drafts, prefilled forms, in-progress notes, context references (`_shared/`).
**What does NOT live here:** Blank templates (→ `04-applications/`), submitted records (→ `08-submitted/`).
