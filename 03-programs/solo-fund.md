# Solo Fund (Solo Founders Program / SFP)

- **url:** https://solofounders.com (or similar SFP landing)
- **type:** accelerator (3-month in-person program)
- **form host:** SFP apply form
- **geo focus:** San Francisco (in-person required)
- **stage:** Building an MVP / Launched with early users / etc.
- **status:** cohort-based (kickoff 2026-05-22)
- **housing:** SFP house option ($1,500/mo, private bedroom above office) or arrange own

## Description

3-month in-person program for solo founders in San Francisco. Strict
in-person requirement (no remote). Housing available on-site for
selected participants.
