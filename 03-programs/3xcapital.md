## Description

Investment fund taking startup submissions via a Tally form. Form
includes typical raise-stage questions plus a sector-taxonomy field
that defaults to Web3-native categories (DeFi / Infra / Web2-to-Web3 /
Metaverse / GameFi / L1 / L2 / NFT / Other) — worth noting for
non-Web3 founders since the taxonomy doesn't fit cleanly.
