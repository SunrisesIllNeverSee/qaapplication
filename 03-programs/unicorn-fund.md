# NextUnicorn.Fund

- **url:** https://nextunicorn.fund
- **type:** multi-track (VC / Accelerator / Mentorship / Idea2Unicorn / CoFounder Match / StartUp Battle)
- **form host:** Google Forms
- **geo focus:** Global (L1 partners split by region: Asia/ME vs America/Europe)
- **stage:** any
- **status:** rolling

## Description

Multi-program fund with several parallel tracks. Application is a
Google Form; submission is followed by an L1 Calendly meeting, then
(if selected) an L2 partner call, then potential pitch event
participation. Active flow uses WhatsApp for asset exchange.
